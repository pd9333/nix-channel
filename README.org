#+title: nix-channel
* usage
#+begin_src sh :eval no
  nix-channel --add https://github.com/pd9333/nix-channel/archive/trunk.tar.gz pd
#+end_src
* development
** build a package
For example, to build ~gost~:
#+begin_src sh
  nix-build . -A gost
#+end_src
** upgrade a package
For instance, if we want to update ~adguardhome~ to ~0.107.2~:
#+begin_src sh
  niv update adguardhome -a rev=v0.107.2
#+end_src

However, for ~adguardhome~ we need to fix the ~sha256~ in ~nix/sources.json~ with the one of:
#+begin_src sh
  nix-prefetch-url --type sha256 $url
#+end_src
